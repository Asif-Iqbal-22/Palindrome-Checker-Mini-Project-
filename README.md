# Palindrome Checker in C

A console-based program to check if a number or string is a palindrome.

## Features
- Check palindrome for numbers
- Check palindrome for strings
- Option to check multiple inputs
- Exit option

## How to Run
1. Compile the program:

    gcc palindrome.c -o palindrome

2. Run:

    ./palindrome
